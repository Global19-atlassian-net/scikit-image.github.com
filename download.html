Stable release¶

Windows¶

Windows binaries are kindly provided by Christoph Gohlke.

The latest stable release is also included as part of the Enthought Python Distribution (EPD) and Python(x,y).

Linux and OSX¶

scikit-image can be installed from the Python packaging index using

pip install -U scikit-image

or

easy_install -U scikit-image

Latest development version¶

Please refer to the latest build instructions.

To build the development version of the scikit, you need a C/C++ compiler, Python, NumPy and Cython.
