scikits-image: An image processing toolbox for SciPy¶

scikits-image is a collection of algorithms for image processing. It is available free of charge and free of restriction. We pride ourselves on high-quality, peer-reviewed code, written by an active community of volunteers.

Getting Started¶

Filtering an image with scikits-image is easy!

from skimage import data, io, filter

image = data.coins() # or any NumPy array!
edges = filter.sobel(image)
io.imshow(edges)

For more examples, please visit our gallery.
